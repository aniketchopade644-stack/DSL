# 📘 Data Structure Practicals

This repository contains all practical programs implemented as part of the **Data Structures Laboratory**.

---

## 📂 List of Practicals

### 🔹 Practical 1: Marks Analysis

* Average marks calculation
* Highest and lowest marks
* Count absent students
* Most frequent marks

---

### 🔹 Practical 2: Linear & Sentinel Search

* Search roll numbers
* Compare searching techniques

---

### 🔹 Practical 3: Binary & Fibonacci Search

* Efficient searching in sorted list

---

### 🔹 Practical 4: Sorting Techniques (Selection & Bubble)

* Sort student percentages
* Display top 5 scores

---

### 🔹 Practical 5: Quick Sort

* Sorting using divide and conquer
* Display top scores

---

### 🔹 Practical 6: Stack - Parenthesis Checker

* Check valid expression using stack

---

### 🔹 Practical 7: Queue (Simple Queue)

* Job queue simulation
* Add/Delete jobs

---

### 🔹 Practical 8: Circular Queue

* Printer job simulation
* Efficient queue management

---

### 🔹 Practical 9: Linked List

* Club member management system
* Add/Delete members
* Display and count members

---

## 🛠️ Technologies Used

* C++
* Python

---

## 🎯 Objectives

* Understand Data Structures concepts
* Implement searching and sorting algorithms
* Learn stack, queue, and linked list
